# test-learn-schematics

此项目用来测试自定义的Schematics，代码生成规则定义在 learn-schematics 项目中。

**请特别注意：由于@angular/schematics是cli工具的组成部分，它的版本号与cli之间有对应关系。因此，如果你不确定对应关系是什么，请不要修改本项目中的package.json。升级必挂，实测。**
